# Out of Scope

This document enumerates features and behaviors that are explicitly
out of scope for the translator at this stage.

These exclusions are intentional.

---

## App Designer Features

- Thumbnails
- Internal data blocks
- Design‑time metadata not required for execution
- Multiple `UIFigure` instances
- Advanced layout containers (tabs, grids, panels)
- Context menus and toolbars

---

## Live Script Features

- Tasks
- Embedded images
- Rich text formatting
- Output‑driven layout inference
- Multi‑section execution graphs

---

## Runtime Behavior

- Persisted runtime state
- Execution timing semantics
- Performance optimizations

---

## Editing and Repair

- Automatic XML repair
- Recovery from malformed XML
- Back‑conversion to `.mlapp`
